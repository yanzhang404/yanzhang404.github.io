此比较方式就是 a>b <=> ab > ba， 就是将ab连起来比ba长。

同时要证明比较的全序关系，即满足

  • 反对称性，若a <= b 且 b <=a，则a = b

  • 传递性，若a <=b, b <= c, 则 a <= c

  • 完全性，a <= b 或 b <= a
